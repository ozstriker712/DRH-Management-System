# DRH-Management-System
A complete Human Resources Management System (DRH) developed with C# (.NET), Entity Framework 6, SQL Server, and DevExpress.
This system manages the full administrative record of each employee, including all documents throughout their career — such as absences, leaves, and HR actions — in full compliance with the Public Service Human Resources Management.

It provides complete control over employee data, administrative documents, and staff statistics, with options to export lists to Excel or Word and to customize the application’s theme and layout.
The program also includes user permissions, advanced search, statistics, and reporting tools for efficient HR management.

Features:

Developed with C# .NET, Entity Framework 6, SQL Server, and DevExpress.

Manage each employee’s administrative file and personal documents.

Track absences, leaves, and other HR events.

Full CRUD: insert, update, delete, print, and search.

User permission system for secure access control.

Export and organize lists to Excel or Word.

Statistics and reports for HR performance tracking.

Customizable interface and themes.

Fully compliant with the Public Service Human Resources Management.

اسم المشروع: نظام إدارة الموارد البشرية (DRH)

الوصف:
نظام متكامل لإدارة الموارد البشرية تم تطويره بـ C# .NET باستخدام Entity Framework 6 وSQL Server مع واجهة احترافية من DevExpress.
يهدف النظام إلى إدارة الملف الإداري لكل موظف مع كافة الوثائق المتعلقة بمساره المهني مثل الغيابات والعطل والتنقلات،
وذلك بما يتوافق مع القانون الأساسي العام للوظيفة العمومية الجزائري.

يوفر النظام أدوات شاملة لإدارة البيانات، التحكم في الصلاحيات، البحث المتقدم، وإعداد الإحصائيات والتقارير،
مع إمكانية تصدير القوائم إلى Excel أو Word، وتخصيص مظهر البرنامج وثيمه حسب رغبة المستخدم.

الميزات:

مطور بـ C# .NET و Entity Framework 6 و SQL Server و DevExpress.

إدارة الملفات الإدارية والوثائق الخاصة بالموظفين.

تتبع الغيابات والعطل بدقة.

عمليات إدخال، تعديل، حذف، طباعة، وبحث.

نظام صلاحيات للتحكم في وصول المستخدمين.

تصدير القوائم إلى Excel أو Word.

إحصائيات وتقارير شاملة لدعم الإدارة.

تحكم في شكل وثيم البرنامج.

متوافق تمامًا مع القانون الأساسي العام للوظيفة العمومية الجزائري.
